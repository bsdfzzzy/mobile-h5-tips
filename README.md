mobile-h5-tips
=============

随着Web前端分享会的深入进行和大家在项目中的不断实践，我们前端开发的同事们积累并成功运用了相当多的前沿技术来使我们的产品达到最优。 本次分享会决定将我们团队的积累整理汇总成《移动端Web应用开发推荐规范》，代表团队的集体智慧，指导和帮助我们海外OA商户进行公众号前端应用开发。同时将在开源社区创建项目，接受广大前端爱好者的批评指正及挑战，不断修正我们的劳动成果，力争打造成移动端Web技术标杆。
